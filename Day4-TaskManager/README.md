#  Day 4 – React Task Manager

A clean and interactive Task Manager built using **React + Vite** as part of my daily React learning journey.

This project focuses on understanding core React concepts like state management, component structure, props, and side effects while also adding smooth UI animations.

---

## Features

- Add new tasks
-  Mark tasks as completed
-  Delete tasks
-  Data persistence using localStorage
-  Smooth CSS animations
-  Clean and responsive UI

---

## Concepts Practiced

- Functional Components
- useState Hook
- useEffect Hook
- Props (Parent → Child communication)
- Conditional Rendering
- Mapping Arrays in React
- CSS Transitions & Animations
- localStorage API

---

## Technologies Used

- React
- Vite
- JavaScript (ES6+)
- CSS3
