# Day 2 - Greeting App 

This project is part of my **React Daily Practice** series. 
A simple React project built with Vite that displays a personalized greeting based on the current time of day. It also includes a **navbar** at the top that shows the current time, updating every second.

---

## Features
- **Dynamic Greeting**: Displays "Good Morning", "Good Afternoon", or "Good Evening" depending on the system time.
- **Props Usage**: Passes a `name` prop to personalize the greeting (e.g., "Good Evening, Devanjali!").
- **Navbar**: A fixed, full-width navigation bar at the top with:
  - Project title
  - Live clock that updates every second
- **Styling**: Fun and colorful design using CSS (centered headings, background highlights, shadows).
- Clean layout using **Flexbox** for centering content

---

##  Tech Stack
- [React](https://react.dev/) (with Vite)
- CSS3 (custom styles)
